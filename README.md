# 🎬 MovieFlix - Movie Database App

A beautiful, responsive movie database application built with vanilla JavaScript and the TMDB API.

## ✨ Features
- ✅ **Trending Movies** - Weekly popular movies
- ✅ **Search** - Find any movie with autocomplete
- ✅ **Movie Details** - Cast, trailers, ratings, overview
- ✅ **Watchlist** - Save movies to watch later
- ✅ **User Ratings** - Rate and review movies
- ✅ **Genre Browsing** - Filter by category
- ✅ **Responsive Design** - Works on all devices

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript (ES6+)
- TMDB API
- Font Awesome Icons
- Local Storage API
